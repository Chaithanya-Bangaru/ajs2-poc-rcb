Proof of Concepts Demo Application using Version 2.0.0-snapshot (30-Sep-2015)

This is a small app used to demonstrate below Angular 2 concepts :

  • Project Setup / Structure

    • IDE Setup
    • Boilerplate Setting files
    • Meaning of SPA - Single Page Application
    • Angular 2 libraries / Packages

  • Introduction to Typescript

    • New Typescript coding style
    • Javascript Object Notation - JSON
    • Java style concepts like interfaces, DTOs,classes, Constructor Injection etc.

  • Data Binding

    • Interpolation - Curly braces {}
    • One way property binding - Square braces []
      • Html Element property binding eg, [src]="expression" ..
      • Angular2 Component property binding eg, <raj-component [vehicle]="expression">
      • Angular2 Directive property binding eg, <div [ngClass]="expression"
      • Attribute binding eg, <button [attr.aria-label]="expression"
      • Class property binding eg, <div [class.isExist]="expression"
      • Style property binding eg, <button [style.color]="expression"
    • Event binding - Round braces ()
    • Two way property binding - Banana in a box [()]

  • Angular 2 Fundamentals

    • Components vs Controllers , Bootstrapping
    • Structural Directives, Dependency Injection, Events
    • Component Lifecycle Hooks
    • Promises vs Observables using rxjs / React extensions
    • Clean Data Services using http, router services
    • Error Handling

  • Filters / Pipes

    • Filtering lists
    • Writing Custom Filters

  • Routing

    • Navigation to different components
    • Passing Parameters to Routes using RouteParams service

  • Micellaneous

    • Nested Components
